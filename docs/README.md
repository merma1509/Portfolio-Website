# Personal Portfolio Website for Mugabo (Niyonshuti Martin)

A modern, responsive portfolio site showcasing expertise in AI/ML, IoT, SpaceTech.

## Structure

- **frontend/**: Next.js application for the website UI.
- **backend/**: FastAPI API for dynamic features (contact forms, blog).
- **docs/**: Documentation and setup guides.

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: FastAPI, PostgreSQL
- Hosting: Vercel/Netlify

## Setup

1. Navigate to `frontend/` and run `npm install`.
2. Navigate to `backend/` and run `pip install -r requirements.txt`.
3. Follow individual READMEs in each folder for detailed instructions.

## Features

- Home, About, Projects, Skills, Contact pages.
- Responsive design.
- Project showcases with images and descriptions.
- Contact form integration.
